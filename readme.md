# Escape Room (Vite + React + TypeScript)

Учебный проект по грейдированию (HTML Academy): каталог квестов, авторизация, бронирование и «Мои бронирования».

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173/`.

Альтернатива: `npm start` (то же самое, что `dev`).

## Скрипты

```bash
npm run lint
npm run build
npm run preview
```

## Структура

- `src/` — исходный код приложения
- `public/` — статика (шрифты, изображения, стили)
- `markup/` — эталонная вёрстка, ui-kit и `sitemap.html`

## Примечания

- `node_modules/` не коммитится и не должен попадать в архив сдачи.
